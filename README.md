RoBoIO Library 1.9
--------------------

RoBoIO is an I/O library for the "RoBoard" robot controller of DMP Electronic 
Inc. It can be compiled on DOS, Linux, Windows CE, and Windows XP. Here is only 
the newest code base of RoBoIO. For complete packages, please download from 

		<http://www.roboard.com/download_ml.htm>

---------------------------------------

#### About RoBoard ####

The RoBoard is an embedded board for Robotics applications that runs an x86 
SoC (Vortex86DX) at 1GHz. It features PWM interfaces, RS-232, RS-485, TTL COM, 
A/D ports, SPI/I2C, USB, LAN and comes with 256MB DDR2 memory and runs off an 
SD Card that serves as a Hard Drive.

RoBoard has been employed in various Robotics projects. You may see interesting 
application videos on RoBoard website:

		<http://www.roboard.com/video01.htm>
